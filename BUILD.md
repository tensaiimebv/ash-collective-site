# Инструкция по сборке

## Проблема

Из-за ограничений сети в текущей среде, зависимости npm не могут быть установлены автоматически.

## Решение

### Вариант 1: Сборка на вашем компьютере

1. Скопируйте папку `ash-collective-site` на свой компьютер
2. Убедитесь, что у вас установлен Node.js 18+
3. Выполните:

```bash
cd ash-collective-site
npm install
npm run build
```

4. Загрузите содержимое папки `dist/` на хостинг (Vercel, Netlify, GitHub Pages)

### Вариант 2: Использование Codesandbox / StackBlitz

1. Загрузите проект на Codesandbox или StackBlitz
2. Они автоматически установят зависимости
3. Соберите проект и скачайте `dist/`

### Вариант 3: Деплой через Git

1. Загрузите проект на GitHub
2. Подключите репозиторий к Vercel или Netlify
3. Они автоматически соберут проект при пуше
